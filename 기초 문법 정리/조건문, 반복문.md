### ✔️ if문

```csharp
			if ( 여기 내용이 참일 경우 )
				{
					내용 실행
				}

			else if (위가 참이 아니면)
				{
					내용 실행
				}

			else (전부 아니면)
				{
					내용 실행
				}

```
- 주의사항으로 `if`문 다음 `if`문이 올 때는 둘다 참인 경우 둘다 실행됨.
- 그래서 안하고 싶으면 `else if`를 해야함.
- `else`는 바로 위의 `if`만 제외됨 -> 더 먼저 나온 `if`의 경우 참일 경우 중복 실행됨.

```csharp
			Console.Write("숫자를 입력하세요: ");
			string input = Console.ReadLine();

			int inputValue;
			bool isConverted = int.TryParse(input, out inputValue);
			//TryParse에서 정수형이 된 정보가 out을 통해 해당 변수로 초기화되는것.
            
			if (isConverted)
			{
				Console.WriteLine($"정수 변환 성공: {inputValue}");
			}

			else
			{
			Console.WriteLine("입력값을 정수로 변환할 수 없습니다.");
				
			}
```

### ✔️ switch문

```csharp
			switch (inputValue)
			{
				case 1:
					Console.WriteLine("상점");
					break;
				case 2:
					Console.WriteLine("마을");
					break;
				default: //if의 else 역할
					Console.WriteLine("아무것도 안함");
					break;
			}
```
- `switch`문은 `break` 필수.

- `if`문과 `switch`문 적절히 골라 사용.
- `switch`문은 `case`를 **숫자나 값으로 할때 정리**를 편하게 할 수 있음.
- **상황을 가정으로 해야하는 경우** `if`문으로 나타내는게 보기 쉬움.

### ✔️ while문

```csharp
   			string input;
			int inputValue;
			bool isConverted = false;

			while (!isConverted)
			{
				Console.Write("정수 입력 : ");
				input = Console.ReadLine();
				isConverted = int.TryParse(input, out inputValue);
				
				if (isConverted)
				{
					Console.WriteLine("정수 입력 성공");
				}
					
			}
```

#### 특정 횟수 반복 while문

```csharp
			int count = 0;
			int toLoop = 10;

			while (count < toLoop)
			{
				count++;

				if (count % 2 == 0) //짝수일 경우만 출력
				{
					Console.WriteLine($"반복문 {count}회 루프 수행중...");
				}

			}
            Console.WriteLine("끝");
```

#### do-while  (일단 하고 반복할지 결정) 한번은 실행되어야 할 때 사용

```csharp
			bool isLoop = false;
			
			do
			{
				Console.WriteLine("do-while 반복");
			} while (isLoop);
```
#### break와 continue

```csharp
			while (true)
			{
				Console.Write("정수 입력 : ");
				input = Console.ReadLine();
				isConverted = int.TryParse(input, out inputValue);
				
				if (inputValue == 0) //0을 입력하면 종료처리
				{
					Console.WriteLine("0 입력, 반복문 종료");
					break;
				}
				
				else if (isConverted)
				{
					Console.WriteLine("정수 입력 성공");
				}
				
				else
				{
					continue;
				}
				
					
			}
```
- `continue` 는 이후나온 코드 수행 없이 **다시 반복문 수행**
- `break`는 반복문을 **끝내고 탈출**

### ✔️ for문

```csharp
			for (초기식; 조건식; 증감식)
            {
            	반복하는 내용
            }
```
- 예시
```csharp
			for (int i = 0; i < 10; i++)
			{
				Console.WriteLine($"반복 {i} 회 실행");
			}
```

#### 반복문 속 반복문 (외부 1번 후 내부 10번 반복 후 외부로)

```csharp
			for (int i = 0; i < 10; i++)
			{
				Console.WriteLine("외부 반복문");
				for (int j = 0; j < 10; j++)
				{
					Console.WriteLine("내부 반복문");
				}
			}
```
#### for 트리 만들기
```csharp

			int floor = 15;

			for (int i = 1; i <= floor; i++)
			{
				for (int j = 1; j <= floor-i; j++)
				{
					Console.Write(" ");
				}

				for (int j = 1; j <= (2 * i - 1); j++)
				{
					Console.Write("*");
				}
				
				Console.WriteLine();
			}
```
