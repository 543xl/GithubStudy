- string 문자열, int 정수, float/double 실수, 문자하나 char, 참거짓 bool.

형변환에서 float의 경우는 앞에 (float)을 붙이거나 숫자 뒤에 f를 붙임.
실수 나눗셈 할 때 몫만 나오는 경우가 있음. 자료형을 정확히 하고 가기.

```csharp
float a = 3f;
float a = (float)4;
```

- 연산자는 사칙연산 + 비교 까지.

```csharp
a = a + b; 	// a += b; 와 동일
a = a * b;
a = a - b;
a = a / b;
a = a % b; 	//나머지

result = a == b; 	//같음
result = a != b; 	//다름
result = a >= b; 	//이상
result = a > b;  	//초과
```
- 증감 연산자 

```csharp
int count = 1;
count++; 			//1씩 더해짐 (count--; 는 1씩 빼기)
```
- 전위 연산, 후위 연산

```csharp
Console.WriteLine(count++) 	//출력하면 1로 나온 후 1이 더해져서 2가 된 상태.
Console.WriteLine(++count) 	//출력하면 3으로 나옴.
```
연산자 위치에 따라 연산이 되고 출력되는지 안되는지 판별
count=count++ 일 경우 count에 1이 더해지는 것이 아닌 기존 count 값을 유지

- NOT은 !, OR은 |, AND는 & 

비트 연산자는 속도가 빠름 0과 1로 나타내서 정보를 빠르게 계산.

status |= 1;        

status = 0000
1      = 0001 			 ->          status = 0001


||, && 는 조건부 논리 연산자. 이대로 하면 앞부터 연산해서 틀리면 뒤 연산 수행 X

논리 연산자는 ||, && 를 쓴다.
```csharp
bool a = true;
bool b = false;

bool result = a|b; //result 출력하면 true
```
